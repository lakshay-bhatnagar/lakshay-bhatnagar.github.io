## Welcome to My Blog! 👋

Hi! I'm **Lakshay Bhatnagar**, a cybersecurity enthusiast and web application security researcher.  
This blog is a collection of my **PortSwigger Web Security labs writeups**, security experiments, and tutorials.

---

## What You’ll Find Here

- 🛡 **Web Security Labs** – Step-by-step writeups for **Broken Access Control, Authentication Vulnerabilities, XSS**, and more.  
- 🧰 **Tools & Techniques** – Insights on tools like **Burp Suite, OWASP ZAP**, and other pentesting utilities.  
- 📚 **Learning Notes** – My personal notes, observations, and learning tips for aspiring security researchers.  

---

## Why This Blog Exists

The goal of this blog is to:

1. **Share knowledge** – Helping others understand security vulnerabilities in a structured way.  
2. **Document progress** – Keep track of my learning journey in web application security.  
3. **Encourage practice** – Inspire others to safely experiment with labs and hands-on exercises.
